## Présentations & marketing

- `poster/` : poster marketing (format A1, Canva/Figma).
- `pitch_commercial/` : diaporama orienté clients (valeur, bénéfices, ROI).
- `pitch_technique/` : présentation avant-vente (architecture, sécurité, intégrations).
- `soutenance/` : présentation détaillant la réalisation SAE.

Chaque support pointera vers des ressources cloud (Google Slides/Canva) en mode suggestion pour traçabilité.
